
# *Jeu d'images cachées à thème de Gundam*


## Pour installer:

### Sur Powershell :

- Clonez le projet sur votre ordinateur
- Rendez-vous à la racine du projet
- Exécutez la commande suivante :
    ```
    .\mvnw.cmd javafx:run 
    ```

### Sur IntelliJ :
- Ouvrez IntelliJ et cliquer sur *Clone Repository*
- Choisissez **Git** comme version de contrôle et inscrivez https://github.com/AlexStarlight03/HiddenPictures comme **URL**. *(Assurez vous de choisir l'emplacement et de nommé un nouveau dossier)*
- Cliquez sur *Clone*
- Une fois le projet ouvert, ouvrez le fichier *src/main/java/com/alexdube/hiddenpictures/**HiddenObjectsApp.java*** dans l'éditeur.
- Cliquez sur la **flèche verte** au dessus de l'éditeur pour lancer le projet.
